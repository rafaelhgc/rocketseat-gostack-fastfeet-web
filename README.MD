<h3>💽 Instalação</h3>

```sh
# Clone o repositório para sua maquina
$ git clone git@github.com:rafaelhgc/rocketseat-gostack-fastfeet-web.git fastfeet-web

# Instale as dependências do projeto
$ cd fastfeet-web
$ yarn
```

<h3>🚀 Iniciando a aplicação</h3>

```sh
# Inicie a aplicação
$ yarn start
```
